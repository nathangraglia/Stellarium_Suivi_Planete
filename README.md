# Stellarium_Suivi_Planete
Petit projet réalisé pour l'UE "Histoire de l'astronomie" au semestre d'automne de l'année 2025-2026.

Il permet d'automatiser la phase d'observation, de récolte des données et d'analyse de ces dernières en donnant des informations utiles pour pratiquer la méthode babylonnienne (les stations, les vitesses max et min, etc...)

Merci de créditer le projet si vous l'utilisez pour faire vos observations.
